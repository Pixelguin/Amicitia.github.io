Setting Up PCSX2
PCSX2 is a cross-platform PS2 emulator.
While it's easy to install, choosing the right settings
can not only enhance your experience with the game on PC,
(making it even more graphically impressive than on native PS2 hardware),
but improve the presentation of your mods!
Use this PS2 BIOS to get started.

Navigate to Config > Emulation Settings > Aspect Ratio
to try widescreen!
Next, try going to Config > Emulation Settings > Speedhacks
Leave them low and disable mVU flag hack if you don't need the speed boost.
In Config > Video > Plugin Settings,
I recommend using a graphics card/hardware renderer
if you have one. Also, Direct3D9 or Direct3D11
over OpenGL. Reasoning: OpenGL fixes shadow draw distance,
but causes clipping problems with close together
textures (rugs, roads, lighting effects...)
Also, Direct3D11 has some occasional flickering.
Direct3D9 does not seem to have any of these problems.

DISABLE native PS2 resolution!! The game will look a lot better.
Anisotropic filtering and texture filtering make
the graphics look a lot smoother, too!
