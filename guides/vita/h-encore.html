Exploiting a 3.67/3.68 Vita

What's Henkaku?

The Playstation Vita's first native exploit, Henkaku, was first made available for firmware 3.60. With it, you could install Vitashell (a powerful homebrew app for installing other homebrew and managing the filesystem). You could also install Henkaku Enso, a bootloader exploit that runs henkaku at startup automatically. At the time of writing, the latest firmware is 3.68.

What's h-encore?

If your Vita is on 3.65, 3.67 or 3.68, you can make use of TheFloW's latest h-encore exploit. This will install a bubble on your livearea that will load Henkaku whenever you run it. Unfortunately, Henkaku Enso will only work on 3.65 using this. On the plus side, all homebrew and plugins that don't require Enso still work, and you can play games that require a newer firmware.

Choosing the right firmware

In the event that a firmware higher than 3.68 is released, do NOT update. It's extremely likely that all currently available homebrew-enabling exploits will be patched, and you will have no alternatives for the forseeable future.
In this case, you may want to buy another system. It's unlikely that a new unit will ship with the very latest firmware, so you'll be able to follow these directions to update to the one of your choice.
If you want Henkaku Enso, go with either 3.60 or 3.65 and use the regular Henkaku exploit to install it. Or, update to 3.68 and follow the rest of this guide on setting up h-encore.

Installing h-encore
Download auto h-encore and extract the ZIP.
Also install and set up QCMA on your computer.

This is an open-source alternative for Sony's Content Manager Assistant app, which forces you to update to the latest firmware to use. Since we want to remain on our desired firmware, this will provide only the functionality that we need. Once it's installed, connect your Vita to your PC (with ethernet if it's a PSTV, or the USB cord it comes with if it's a portable model). You may also connect with Wi-fi if you're on 3.68 and if it's still the latest firmware (which it is at the time of writing). Otherwise you may need to look into other ways to block the firmware update prompt.

The first time you run QCMA, the settings window will automatically open. You should configure it like the screenshot on the right.
When you connect to your PC for the first time, it should prompt you to enter a PIN as it appears on your screen. Once you're connected, on your Vita navigate to Content Manager > Copy Content > System to PC > Applications. There's a chance you may get an error message (C0-14258-0), in which case fully rebooting the Vita and starting over usually resolves the issue.
If you manage to reach this screen, feel free to back out of the Content Manager app for now and go back to your PC. We don't actually need to transfer a file yet, but we did need to generate a folder with your Account ID. The latest version of the auto h-encore installer should be able to locate this automatically if you did the previous steps.
Press Start and wait for it to automatically setup a h-encore application that will work on your account.
When the process is complete, a window will pop up with directions for you to follow. This time, let's transfer the application from our PC to the Vita using the Content Manager. Before we do so, find the QCMA icon in your taskbar, right click it, and choose Refresh Database so that the Vita can find the new files.
If you followed all of the steps correctly, the h-encore bubble should transfer to the Vita live area with no issues! If not, go back and make sure your account ID is correct or try rebooting. As long as you do NOT update past 3.68, you will be able to launch this bubble anytime you want in order to enable homebrew. It will persist until the next time you reboot, in which case you have to run the h-encore bubble again. Still, it's very quick and painless to do each time.

What you can do with h-encore
Right away, you have the option to install Vitashell from the h-encore bubble, and it's highly recommended that you do so. Vitashell is your ticket to installing homebrew (VPK files downloaded from the internet), transferring files over Wi-fi or ethernet using FTP (file transfer protocol using a program like Filezilla-- no need for the content manager anymore!), or installing plugins.
Press Select while in Vitashell to start the FTP server. Enter the IP and port on PC and you'll be able to send files to and from both ends of the server!
If you're interested in backing up, decrypting (for editing), or transferring save files from another system, I recommend installing Vita-Savemgr. Backed up saves will be transferred to ux0:data/savegames/TITLEID/, where you can send them to PC over FTP using Vitashell if you want.
If you'd like to run mods for games and you're using h-encore, you need to follow this guide on installing and using rePatch.
By running gamesd.skprx alongside an unofficial adapter, you can mount your MicroSD card as a replacement for the system's memory card partition (ux0). Read this guide for more details.
