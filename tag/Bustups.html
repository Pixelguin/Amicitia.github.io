BustupParamEditorGUI
P5 tool by ShrineFox
11/15/2019
New version of BustupParamEditor that allows you to change the values for X and Y positioning of the mouth, eyes, or the entire bustup. You can also add/rename entries, create new bustup .bin files, and repack edited images back into P5's bustup .bin format.
Update 11/15/2019:
- Allow adding/renaming entries, creating new .bin files
- Automatically tries to recover last param/bustup dir path at startup
- Uses values from param file to select bustups now instead of static list of names
- Appends new entries to selected params file
- Can now create .bins for param entries that lack one
- Can now rename existing/unused entries instead of adding them
